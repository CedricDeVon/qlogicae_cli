<h2 id="about">
  📖 About
</h2>
<h3 id="about-description">
  🧾 Description
</h3>
<p>
  <strong>QLogicae CLI</strong> is a multi-purpose C++ software development tool with a purpose - make C++ development great again (as if it always was).
  
</p>
<h3 id="about-core-features">
  ⚙️ Core Features
</h3>
<ul>
  <li>
    <h3>QLogicae Project Manager</h3>
    <ul>
      <li><p style="font-weight: bold">Public and Private Configuration Systems</p>
      <li><p style="font-weight: bold">Multi-Environment Build Systems</p>
      <li><p style="font-weight: bold">Windows Registry Handling</p>
      <li><p style="font-weight: bold">Environment Variable Handling</p>
      <li><p style="font-weight: bold">Logging Systems</p>
      <li><p style="font-weight: bold">And More</p>
    </ul>
  </li>
  <li>
    <h3>QLogicae-VS2022 Integrated Automation</h3>
    <ul>
      <li><p style="font-weight: bold">QLogicae Opera Integrated Setup</p>
      <li><p style="font-weight: bold">Visual Studio 2022 Build Process</p>
      <li><p style="font-weight: bold">Application Deployment Via Inno Setup</p>
      <li><p style="font-weight: bold">And more</p>
    </ul>    
  </li>
</ul>

</h2>
<h3 id="getting-started-pre-requisites">
  📋 Pre-requisites
</h3>
<p>
  In order to ensure the most stable user-experience as possible, please meet the requirements below:
</p>
<ul>
  <li><p>OS - Windows 10 (Or Higher)</p></li>
  <li><p>System Architecture - x64</p></li>
  <li><p>Minimum Storage - At least 20 MB</p></li>
  <li><p>Minimum RAM - At least 1 GB</p></li>
  <li><p>PowerShell - 7.5.3.0 (Or Higher)</p></li>
</ul>
